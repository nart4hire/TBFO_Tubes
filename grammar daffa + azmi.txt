S ->  DEF | IMPORT | CLASS | SUITE | WITH

SUITE -> IF | LOOP | ASSIGNMENT | METHOD

ASSIGNMENT -> Var Assign EXPR

EXPR -> Var | Var Op EXPR | Var Per EXPR | Var Comma EXPR | Var Dot METHOD | METHOD | Var Comp EXPR | Po EXPR Pc | Bool | Not EXPR | IN | IS

METHOD -> Var Po EXPR Pc | Var Po EXPR Pc Dot METHOD

IF -> If EXPR End | If EXPR End NL Raise Var | If Ex End NL ELIF NL ELSE | If Ex End NL ELSE | If Ex End NL SUITE NL ELIF NL SUITE NL ELSE | If Ex End NL SUITE NL ELSE | If Ex End NL SUITE NL ELIF NL ELSE | If Ex End NL ELIF NL SUITE NL ELSE

ELIF -> ELIF NL ELIF | ELIF NL SUITE NL ELIF | Elif EXPR End

ELSE -> Else End

WHILE -> While EXPR End | While EXPR End NL CONTROL | While EXPR End NL SUITE NL CONTROL

FOR -> For EXPR End | For EXPR End NL CONTROL | For EXPR End NL SUITE NL CONTROL

DEF -> Def Var Po EXPR Pc End | Def Var Po EXPR Pc End NL Pass | Def Var Po EXPR Pc End NL Return EXPR | Def Var Po EXPR Pc End NL SUITE NL Return EXPR

IMPORT -> Import Var | IMPORT As Var | From Var IMPORT

NL -> Newline NL | Newline

CLASS -> Class Var End | Class Var End NL Pass

LOOP -> FOR | WHILE

CONTROL -> Break | Continue

WITH -> With EXPR as EXPR End

IN -> EXPR In EXPR | For EXPR In EXPR End

IS -> Var Is EXPR | Po IS Pc

Comma -> ","
Import -> "import"
Return -> "return"

T -> "True"
F -> "False"

Op -> "+", "-", "*", "/", "//", "%", "**"
Po -> "("
